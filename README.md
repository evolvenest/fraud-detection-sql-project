# Cyber Fraud Detection Analysis (SQL Project)

## Overview
This project analyzes cyber fraud cases using SQL to identify attack patterns, financial impact, and high-risk attack types.

## Tools Used
- MySQL
- Excel (CSV)
- GitHub

## Dataset
Columns used:
- id
- name
- type_of_attack
- amount_loss
- place

## Analysis Tasks
1. Data overview
2. Attack type distribution
3. Financial impact analysis
4. Location-based analysis
5. Risk prioritization

## SQL Concepts Used
- GROUP BY
- COUNT, SUM
- DISTINCT
- ORDER BY
- Window Functions (RANK)

## Conclusion
This project demonstrates practical SQL skills for cybersecurity data analysis.
